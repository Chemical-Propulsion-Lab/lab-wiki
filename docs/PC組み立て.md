# 💻 PC自作ガイド


# 📌 目的
このドキュメントは、PC自作が初めての人でも安全に組み立てられるようにするためのガイドです。

---

## 🧩 1-1 使用パーツ一覧


| パーツ | 製品名 | 説明 |
|--------|--------|------|
| CPU | Intel Core i5-14400 |　パソコン全体の性能を決めるパーツ。多くの計算処理を担当し、人の頭脳にも例えられる。IntelとAMDが２大メーカー。　|
| メモリ | Crusial DDR4 Pro 64GB| CPUが使うデータを一時的に保存しておく場所。作業スペースに例えられる。電源OFFにすると、メモリのデータは消える。　|
| マザーボード | ASUS-PRIME B760M-A D4 | 各パーツをつなぐところ。　|
| CPUクーラー | RT400-ARGB-BK | 発熱するCPUを冷やすためのもの。今回は空冷式を用いる。intelのCPUに付属しているものと、別の箱に入っているものどちらを使ってもよい。好み。　|
| HDD | Seagate BarraCuda 3.5インチ | データを長期間保存するためのもの。電源OFFにしてもデータは残る。同じようなものにSSDというものがあるが、SSDは読み書きが速く、容量当たりの価格が高い。HDDは読み書きが遅いが、安くて大容量。 |
| グラフィックボード | GEFORCE RTX 5060 VENTUS 2X OC | GPU(計算を実行するチップ)を使って大量の計算を超高速で行うためのもの。CPUは少数の複雑な処理が得意だが、GPUは大量の単純計算の並列処理が得意。 |
| 電源ユニット | ELITE GOLD 850 | 電力を供給するためのもの。 |
| PCケース | | パソコンの外装。 |

## 🔧 1-2 必要な道具

- プラスドライバー
- 静電気防止リストバンド（なくてもよい）
- 作業スペース（広くて平らな場所）

---


## ⚠️ 1-3 事前準備

- 金属に触れて静電気を逃がす
- パーツを箱から出して付属品を確認する
- マザーボードの説明書を手元に用意する

---

## 🛠️ 2 組み立て手順

PC組み立てに決まった手順はないが、ある程度のセオリーはある。ここでは、

CPU・CPUクーラー・メモリ取り付け
↓
電源接続 → 動作確認
↓
マザーボードをケースへ取り付け
↓
電源ユニットをケースへ取り付け
↓
配線
↓
グラフィックボード取り付け

の順で解説する。

### CPU・CPUクーラー・メモリ取り付け

基本は以下のサイトを見るとできる↓

[自作パソコンの組立て方(パソコン工房)](https://www.pc-koubou.jp/pc/jisaku_pc.php?srsltid=AfmBOopLNz0wVpn7skWykqhtZSFfOTZfhMK7tRMcQ_XWyrXF_PUXSAE5)

このサイトではIntelのCPUに付属しているCPUクーラーの取り付け方を解説しているが、RT400-ARGB-BKをCPUクーラーに使いたいという人向けに補足をする。

風の流れについて
ケースファンやCPUクーラーは、一般的に支柱がある側が排気、ない側が吸気である。これを意識し、CPUクーラーもケースファンと同じ向きに風が流れるように設置することで熱がこもらないようにする。
