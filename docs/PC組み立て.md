# 💻 PC自作ガイド


# 📌 目的
このドキュメントは、PC自作が初めての人でも安全に組み立てられるようにするためのガイドです。

---

## 🧩 1-1 使用パーツ一覧


| パーツ | 製品名 | 説明 |
|--------|--------|------|
| CPU | |　パソコン全体の性能を決めるパーツ。多くの計算処理を担当し、人の頭脳にも例えられる。IntelとAMDが２大メーカー。　|
| メモリ | | CPUが使うデータを一時的に保存しておく場所。作業スペースに例えられる。電源OFFにすると、メモリのデータは消える。　|
| マザーボード | | 各パーツをつなぐところ。　|
| CPUクーラー | | 発熱するCPUを冷やすためのもの。今回は空冷式を用いる。intelのメモリに付属しているものと、別の箱に入っているものどちらを使ってもよい。好み。　|
| HDD | | データを長期間保存するためのもの。電源OFFにしてもデータは残る。同じようなものにSSDというものがあるが、SSDは読み書きが速く、容量当たりの価格が高い。HDDは読み書きが遅いが、安くて大容量。 |
| グラフィックボード | | GPU(計算を実行するチップ)を使って大量の計算を超高速で行うためのもの。CPUは少数の複雑な処理が得意だが、GPUは大量の単純計算の並列処理が得意。 |
| 電源ユニット | | 電力を供給するためのもの。 |
| PCケース | | パソコンの外装。 |

---


## 🔧 1-2 必要な道具

- プラスドライバー（No.2）
- 静電気防止リストバンド（推奨）
- 作業スペース（広くて平らな場所）

---


## ⚠️ 1-3 事前準備

- 金属に触れて静電気を逃がす
- パーツを箱から出して付属品を確認する
- マザーボードの説明書を手元に用意する

---

## 🛠️ 2 組み立て手順

PC組み立てに決まった手順はないが、ある程度のセオリーはある。ここでは、

[CPU取り付け]
↓
[CPUクーラー・メモリ取り付け]
↓
[電源接続 → 動作確認]
↓
[マザーボードをケースへ取り付け]
↓
[電源ユニットをケースへ取り付け]
↓
[配線]
↓
[グラフィックボード取り付け]

の順で解説する。
